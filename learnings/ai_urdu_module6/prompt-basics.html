Topic 37

Prompt کی بنیادی سمجھ

AI سے بہتر جواب لینے کا پہلا قدم واضح سوال اور صاف ہدایت ہے۔

مختصر تعارف:

Prompt وہ ہدایت، سوال یا معلومات ہیں جو آپ LLM کو دیتے ہیں تاکہ وہ آپ کے مقصد کے مطابق جواب بنا سکے۔

Prompt کیا ہے؟

Prompt صرف ایک سوال نہیں، بلکہ ایک مکمل ہدایت ہو سکتی ہے۔ مثال کے طور پر آپ AI سے کہہ سکتے ہیں کہ وہ کسی موضوع کو آسان زبان میں سمجھائے، کسی email کو بہتر بنائے، کسی plan کو steps میں لکھے، یا کسی text کا خلاصہ دے۔

LLM آپ کے الفاظ، context اور مطلوبہ output کو دیکھ کر جواب بناتا ہے۔ اگر prompt مبہم ہو تو جواب بھی عام، ادھورا یا غیر متعلق ہو سکتا ہے۔

اچھا prompt کیوں ضروری ہے؟

اچھا prompt AI کو سمت دیتا ہے۔ اس سے جواب میں غیر ضروری تفصیل کم ہوتی ہے، انداز بہتر ہوتا ہے، اور نتیجہ آپ کے کام کے زیادہ قریب آتا ہے۔

اچھا prompt وقت بھی بچاتا ہے، کیونکہ آپ کو بار بار وضاحت نہیں دینی پڑتی۔

کمزور اور بہتر مثال

کمزور prompt:

میرے لیے marketing لکھو۔

بہتر prompt:

ایک چھوٹے online کپڑوں کے کاروبار کے لیے Instagram post لکھیں۔ مقصد نئی summer collection معرفی کروانا ہے۔ tone دوستانہ ہو، text مختصر ہو، اور آخر میں action line شامل کریں۔

دوسری مثال میں مقصد، کاروبار، platform، tone، لمبائی اور مطلوبہ نتیجہ واضح ہے۔ اسی لیے جواب بہتر آئے گا۔

اہم نکات

  • Prompt AI کو دی گئی ہدایت ہے۔
  • صاف مقصد بہتر جواب کی بنیاد ہے۔
  • Context دینے سے جواب زیادہ متعلقہ بنتا ہے۔
  • مبہم سوال اکثر مبہم جواب دیتا ہے۔
