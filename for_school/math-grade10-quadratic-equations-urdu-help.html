دو درجہ مساوات

جماعت ۱۰ | Quadratic Equations

حصہ ۱: معیاری شکل

ax² + bx + c = 0 جہاں a ≠ 0

دو درجہ مساوات میں x کی سب سے بڑی قوت ۲ ہوتی ہے۔

مثالیں

x² − 5x + 6 = 0 | 2x² + 3x − 2 = 0 | x² − 4 = 0

حصہ ۲: تجزیہ (Factorization)

دو ایسے اعداد تلاش کریں جن کا حاصل ضرب c اور مجموع b ہو۔

x² − 5x + 6 = 0 حل کریں

اعداد: −2 اور −3 (حاصل ضرب = 6، مجموع = −5)

(x − 2)(x − 3) = 0 → x = 2 یا x = 3

حصہ ۳: دو درجہ فارمولا

x = (−b ± √(b²−4ac)) / 2a

2x² + 3x − 2 = 0 (a=2، b=3، c=−2)

D = 9 + 16 = 25 | x = (−3 ± 5)/4

x = 1/2 یا x = −2

حصہ ۴: مميز (Discriminant)

D = b² − 4ac
D کی قیمتجڑوں کی نوعیت
D > 0دو حقیقی اور مختلف جڑیں
D = 0دو برابر حقیقی جڑیں
D < 0کوئی حقیقی جڑ نہیں
💡 جڑوں کا مجموع = −b/a | جڑوں کا حاصل ضرب = c/a

مشق کے لیے تیار ہیں؟
