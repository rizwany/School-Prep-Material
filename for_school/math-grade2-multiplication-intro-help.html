Multiplication Intro — Equal Groups

Multiplication is just a quick way to add the same number over and over. Instead of writing 4 + 4 + 4 = 12, we write 3 × 4 = 12. In this guide you'll learn to think in equal groups, draw arrays, and start using basic times tables.

⏱ ~5 min read 📝 Jump to Worksheet →

Section 1

What is Multiplication?

Multiplication counts equal groups. When every group has the same number of items, instead of adding again and again, we multiply.

We write multiplication using the × symbol: number of groups × size of each group = total.

Example: 3 bags, each with 4 apples → 3 × 4 = 12 apples total.

Reading the symbol: 3 × 4 is read as "3 times 4" or "3 groups of 4".

It doesn't matter which way around you write it: 3 × 4 = 4 × 3 = 12. This is called the commutative property.

Section 2

Three Ways to Think About Multiplication

  1. 1
    Equal groups. Draw circles for groups, dots inside for items. 3 × 4: draw 3 circles, put 4 dots in each → count all dots.
  2. 2
    Repeated addition. Write out the addition: 4 + 4 + 4 = 12. The number of 4s matches the first number (3 fours).
  3. 3
    Array. Arrange dots in rows and columns. 3 rows × 4 columns = 12 dots. Count across a row or down a column to check.

3 rows × 4 columns = 12 dots. This array shows 3 × 4 = 12.

Section 3

Worked Examples

Example 1 — Using repeated addition
Work out 3 × 4.
3 groups of 4 = 4 + 4 + 4
4 + 4 = 8, then 8 + 4 = 12
✅ 3 × 4 = 12
Example 2 — 2 times table
Work out 5 × 2.
5 groups of 2 = 2+2+2+2+2 skip-count by 2s
2, 4, 6, 8, 10
✅ 5 × 2 = 10
Example 3 — Commutative: same answer both ways
Show that 4 × 3 = 3 × 4.
4 × 3 = 3+3+3+3 = 12
3 × 4 = 4+4+4 = 12 same answer!
✅ 4 × 3 = 3 × 4 = 12
Example 4 — Finding a missing factor
4 × ___ = 20. What goes in the blank?
4 × 1=4, 4×2=8, 4×3=12, 4×4=16, 4×5=20 count up
✅ 4 × 5 = 20
Example 5 — Word problem with groups
There are 6 egg boxes. Each holds 2 eggs. How many eggs in total?
6 boxes × 2 eggs each
6 × 2 = 12
✅ 12 eggs

Section 4

Watch Out! ⚠️

Common Mistakes to Avoid

  • ❌ Adding instead of multiplying: Answering 3 × 4 as 3 + 4 = 7
    ✅ Multiplication is repeated addition of the SAME number: 4+4+4=12.
  • ❌ Multiplying by 0 and getting the other number: 5 × 0 = 5
    ✅ Any number × 0 = 0. Zero groups means no items at all.
  • ❌ Mixing up N × 1 with × 0: Saying N × 1 = 0
    ✅ N × 1 = N. One group of N things is still N things.

Section 5

Quick Reference

N × M means
N groups with M items in each group.
Repeated addition
3 × 5 = 5 + 5 + 5 = 15
Commutative
3 × 4 = 4 × 3 (order doesn't change answer)
× 1 rule
Any number × 1 = itself: 7 × 1 = 7
× 0 rule
Any number × 0 = 0: 7 × 0 = 0
Array
Rows × columns = total dots (picture of multiplication)
