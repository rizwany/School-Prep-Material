Addition & Subtraction
3-Digit Numbers

In this guide, you'll learn how to add and subtract 3-digit numbers using the column method — including carrying over and borrowing when needed.

Section 1

3-Digit Numbers & Place Value

A 3-digit number has a Hundreds column, a Tens column, and a Ones column. For example, 347 = 3 hundreds + 4 tens + 7 ones.

When adding or subtracting, we always work right to left — Ones first, then Tens, then Hundreds. This keeps the place values aligned.

If a column's sum is 10 or more during addition, we carry the extra ten to the next column. In subtraction, if a column's top digit is too small, we borrow from the next column.

Section 2

Step-by-Step Method

H T O
Always line up Hundreds, Tens, Ones — then work right to left
  1. 1
    Write numbers in columns. Line up Hundreds, Tens, and Ones carefully so digits are directly above each other.
  2. 2
    Start with Ones. Add (or subtract) the Ones column. If the result is 10 or more (addition), write the Ones digit and carry 1 to the Tens column.
  3. 3
    Move to Tens. Add the Tens digits (including any carried 1). Write the result, carrying if needed.
  4. 4
    Finish with Hundreds. Add the Hundreds digits (including any carried 1). Write the result.
  5. 5
    For subtraction with borrowing: If the top Ones digit is smaller than the bottom, borrow 1 ten from the Tens column — it becomes 10 extra Ones.

Section 3

Worked Examples

Example 1 — Easy Addition (No Carrying)
214 + 153 = ?
Ones: 4 + 3 = 7
Tens: 1 + 5 = 6
Hundreds: 2 + 1 = 3
✅ Answer: 367
Example 2 — Easy Subtraction (No Borrowing)
587 − 243 = ?
Ones: 7 − 3 = 4
Tens: 8 − 4 = 4
Hundreds: 5 − 2 = 3
✅ Answer: 344
Example 3 — Addition with Carrying
456 + 378 = ?
Ones: 6 + 8 = 14 → write 4, carry 1
Tens: 5 + 7 + 1(carry) = 13 → write 3, carry 1
Hundreds: 4 + 3 + 1(carry) = 8
✅ Answer: 834
Example 4 — Subtraction with Borrowing
523 − 167 = ?
Ones: 3 < 7, borrow 1 ten → 13 − 7 = 6
Tens: 2 − 1(borrowed) = 1; 1 < 6, borrow → 11 − 6 = 5
Hundreds: 5 − 1(borrowed) − 1 = 3
✅ Answer: 356
Example 5 — Word Problem
A school library has 342 books. It receives 219 new books. How many books are there now?
342 + 219 = ?
Ones: 2+9=11 → write 1, carry 1
Tens: 4+1+1=6
Hundreds: 3+2=5
✅ Answer: 561 books

Section 4

Watch Out! ⚠️

Common Mistakes to Avoid

  • ❌ Misaligning columns: Adding hundreds to tens or tens to ones
    ✅ Always line up the digits by place value before calculating
  • ❌ Forgetting the carry: Writing 14 in the ones column instead of 4 (carrying 1)
    ✅ When a column totals 10+, write the units digit and write a small "1" above the next column
  • ❌ Subtracting the wrong way in a column: Doing 3 − 7 as if it's 7 − 3 = 4 without borrowing
    ✅ If the top digit is smaller, you must borrow from the next column before subtracting

Section 5

Quick Reference

Always work: Ones → Tens → Hundreds
Carry (Addition)
When a column sum ≥ 10, carry the tens digit to the next column left.
Borrow (Subtraction)
When top digit < bottom digit, borrow 1 from the next column, adding 10 to current column.
Hundreds
The leftmost digit in a 3-digit number. Worth 100 each.
Tens
Middle digit. Worth 10 each.
Ones
Rightmost digit. Worth 1 each.
Check Addition
Subtract one addend from the sum — you should get the other addend.
