Real Numbers

Grade 9 Math  ·  Number systems, irrationals & surds

🔢 The Number Hierarchy

Every number fits into a nested system. Each set contains all the sets above it.

Natural (ℕ)
Counting numbers starting from 1.
1, 2, 3, 4 …
Whole (W)
Natural numbers plus zero.
0, 1, 2, 3 …
Integers (ℤ)
Whole numbers and their negatives.
… −2, −1, 0, 1, 2 …
Rational (ℚ)
Any number expressible as p/q where p, q ∈ ℤ and q ≠ 0. Decimal terminates or repeats.
½, 0.75, −3, 0.3̄
Irrational
Cannot be written as p/q. Decimal is non-terminating and non-repeating.
√2, π, √3, e
Real (ℝ)
ALL rational AND irrational numbers together form the real number system.
any point on the number line

√ Surds (Irrational Roots)

A surd is a root that cannot be simplified to a rational number: √2 ≈ 1.4142… (non-terminating, non-repeating).

Note: √4 = 2 is not a surd because it simplifies to a whole number.

√(ab) = √a × √b
√12 = √4 × √3 = 2√3
√(a/b) = √a / √b
√(9/25) = 3/5
a√n + b√n = (a+b)√n
3√5 + 2√5 = 5√5
√n × √n = n
√7 × √7 = 7

✏️ Worked Examples

1

Classify: −4, ½, √5, 0, π

1

−4 is an integer (and rational). ½ is rational (p/q form).

2

√5 ≈ 2.2360… non-terminating, non-repeating → irrational.

3

0 is a whole number (and rational). π ≈ 3.14159… → irrational.

Rational: −4, ½, 0  |  Irrational: √5, π
2

Simplify √75

1

Find the largest perfect-square factor of 75: 75 = 25 × 3.

2

√75 = √(25 × 3) = √25 × √3 = 5√3.

√75 = 5√3
3

Simplify 3√8 − √2

1

Simplify √8: √8 = √(4×2) = 2√2.

2

3√8 = 3 × 2√2 = 6√2.

3

6√2 − √2 = (6−1)√2 = 5√2.

3√8 − √2 = 5√2
4

Is 0.121212… rational or irrational?

1

The decimal pattern "12" repeats endlessly → it is a recurring decimal.

2

Any terminating or recurring decimal can be expressed as p/q, so it is rational.

3

0.12̄ = 12/99 = 4/33.

0.121212… = 4/33 → Rational
5

Simplify √18 × √2

1

Combine under one radical: √18 × √2 = √(18 × 2) = √36.

2

√36 = 6 (a perfect square → rational).

√18 × √2 = 6

⚠️ Watch Out!

  • SURD CHECK√9 = 3 is rational, NOT a surd. Always check if the result simplifies to a whole number.
  • LIKE SURDSYou can only add/subtract surds with the same radicand: 2√3 + 5√3 = 7√3, but 2√3 + 5√2 cannot be simplified further.
  • π ≠ 22/722/7 is only an approximation of π. π itself is irrational — its decimal never repeats.
  • PRODUCTThe product of two irrationals can be rational: √2 × √2 = 2.
