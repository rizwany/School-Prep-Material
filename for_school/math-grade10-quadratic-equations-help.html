Quadratic Equations

Grade 10 Math  ·  Factoring, formula & the discriminant

📐 Standard Form

A quadratic equation has the form ax² + bx + c = 0, where a ≠ 0. It is a polynomial equation of degree 2 and always has exactly 2 roots (counting multiplicity, possibly complex).

The roots (solutions) are the x-values that make the equation true. Geometrically, they are the x-intercepts of the parabola y = ax² + bx + c.

🔧 Methods of Solving

Factoring
ax²+bx+c = (px+q)(rx+s)
Fast when factors are obvious integers.
Quadratic Formula
x = (−b ± √(b²−4ac)) / 2a
Always works. Use when factoring is hard.
Completing the Square
(x + b/2a)² = (b²−4ac)/4a²
Useful for deriving the formula or vertex form.
Square Root Method
x² = k → x = ±√k
Only when equation has no x term (b = 0).

△ The Discriminant (D = b² − 4ac)

D > 0
Two distinct real roots
D = 0
One repeated real root
D < 0
No real roots (complex)

Sum of roots: α + β = −b/a  |  Product of roots: αβ = c/a

✏️ Worked Examples

1

Solve x² − 5x + 6 = 0 by factoring.

1

Find two numbers that multiply to 6 and add to −5: they are −2 and −3.

2

Factor: (x − 2)(x − 3) = 0.

3

x − 2 = 0 → x = 2, or x − 3 = 0 → x = 3.

x = 2 or x = 3
2

Solve 2x² + 3x − 2 = 0 using the quadratic formula.

1

a = 2, b = 3, c = −2. D = 3² − 4(2)(−2) = 9 + 16 = 25.

2

x = (−3 ± √25) / 4 = (−3 ± 5) / 4.

3

x = (−3+5)/4 = 1/2 or x = (−3−5)/4 = −2.

x = ½ or x = −2
3

Find the discriminant of x² + 4x + 4 = 0 and classify the roots.

1

D = 4² − 4(1)(4) = 16 − 16 = 0.

2

D = 0 means one repeated root. x = −4/2 = −2.

D = 0; repeated root x = −2
4

The roots of x² − px + 12 = 0 are 3 and 4. Find p.

1

Sum of roots = 3 + 4 = 7 = p/1 → p = 7.

2

Check product: 3 × 4 = 12 = c/a ✓.

p = 7
5

Solve x² − 8 = 0.

1

x² = 8 → x = ±√8 = ±2√2.

x = 2√2 or x = −2√2

⚠️ Watch Out!

  • SIGNAlways rearrange to ax²+bx+c = 0 before applying the formula. Don't forget to set equal to zero first.
  • DIVIDEThe full formula is (−b ± √D) / (2a) — the 2a divides the entire numerator, not just ±√D.
  • TWO ROOTSEven if factoring gives the same bracket twice, e.g. (x−3)² = 0, there are still two roots — they're just equal (x = 3 twice).
  • a ≠ 0If a = 0 the equation becomes linear (bx + c = 0), not quadratic.
