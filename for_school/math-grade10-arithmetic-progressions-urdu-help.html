حسابی تسلسل

جماعت ۱۰ | Arithmetic Progressions (AP)

حصہ ۱: حسابی تسلسل کیا ہے؟

وہ تسلسل جس میں ہر دو مسلسل اعداد کا فرق ایک جیسا ہو — اسے حسابی تسلسل کہتے ہیں۔

a, a+d, a+2d, a+3d, … جہاں a = پہلی حد، d = عام فرق

مثال: 2، 5، 8، 11، 14، …

a = 2 (پہلی حد) | d = 3 (عام فرق = 5−2 = 3)

حصہ ۲: n ویں حد کا فارمولا

aₙ = a + (n−1)d

2، 5، 8، … کی 10ویں حد = ؟

a₁₀ = 2 + (10−1)×3 = 2 + 27 = 29

💡 n ویں حد معلوم کرنے کے لیے: a + (n−1)d میں a، d اور n ڈالیں۔

حصہ ۳: پہلی n حدوں کا مجموع

Sₙ = n/2 × [2a + (n−1)d] یا Sₙ = n/2 × (a + l)

جہاں l = آخری حد (nth term)

2، 5، 8، … کی پہلی 10 حدوں کا مجموع = ؟

S₁₀ = 10/2 × [2(2) + (10−1)×3] = 5 × [4 + 27] = 5 × 31 = 155

حصہ ۴: فارمولے کا جدول

چیزفارمولا
n ویں حدaₙ = a + (n−1)d
عام فرقd = a₂ − a₁
مجموع (d کے ساتھ)Sₙ = n/2 × [2a + (n−1)d]
مجموع (آخری حد کے ساتھ)Sₙ = n/2 × (a + l)

مشق کے لیے تیار ہیں؟
