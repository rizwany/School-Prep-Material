Exponents (Laws)

Grade 8 Math  ·  Laws of indices and applications

💡 What is an Exponent?

In aⁿ, a is the base and n is the exponent (or index/power). It means "multiply a by itself n times": 2⁴ = 2×2×2×2 = 16.

📜 The Laws of Exponents

Product Rule
aⁿ × aᵐ = aⁿ⁺ᵐ
2³ × 2² = 2⁵ = 32
Quotient Rule
aⁿ ÷ aᵐ = aⁿ⁻ᵐ
3⁵ ÷ 3² = 3³ = 27
Power Rule
(aⁿ)ᵐ = aⁿˣᵐ
(2³)² = 2⁶ = 64
Zero Exponent
a⁰ = 1
7⁰ = 1 (a ≠ 0)
Negative Exponent
a⁻ⁿ = 1/aⁿ
2⁻³ = 1/8
Power of Product
(ab)ⁿ = aⁿbⁿ
(2×3)² = 4×9 = 36

✏️ Worked Examples

1

Simplify 3⁴ × 3³

1

Same base → add exponents: 3⁴ × 3³ = 3⁴⁺³ = 3⁷

3⁷ = 2187
2

Simplify 5⁶ ÷ 5²

1

Same base → subtract exponents: 5⁶ ÷ 5² = 5⁶⁻² = 5⁴

5⁴ = 625
3

Simplify (2³)⁴

1

Power rule → multiply exponents: (2³)⁴ = 2³×⁴ = 2¹²

2¹² = 4096
4

Evaluate 4⁻²

1

Negative exponent → reciprocal: 4⁻² = 1/4² = 1/16

4⁻² = 1/16 = 0.0625
5

Simplify (x²y³)⁴

1

Power of product: raise each factor to the power of 4.

2

(x²)⁴ = x⁸;   (y³)⁴ = y¹²

(x²y³)⁴ = x⁸y¹²

⚠️ Watch Out!

  • SAME BASEProduct and quotient rules only work when the bases are identical: 2³ × 3² ≠ 6⁵.
  • ZERO EXPa⁰ = 1 for any non-zero a. 0⁰ is undefined.
  • NEG BASE(−2)⁴ = 16 (even power → positive). (−2)³ = −8 (odd power → negative).
  • ADD vs MULT2³ + 2³ = 2×2³ = 2⁴, but 2³ × 2³ = 2⁶. Adding is NOT the same as multiplying.
