Statistics (Advanced)

Grade 10 Math  ·  Grouped data, mean methods, median & mode

📊 Measures of Central Tendency

Mean (x̄)
Sum of all values divided by the number of values.
Median
Middle value when data is arranged in order. For even n: average of two middle values.
Mode
Value that appears most frequently. A dataset can have no mode, one mode, or multiple modes.

For grouped data, we use class intervals and midpoints (class marks) to estimate these measures.

📐 Mean of Grouped Data — Three Methods

Direct Method
x̄ = Σ(fᵢxᵢ) / Σfᵢ
xᵢ = midpoint of class, fᵢ = frequency. Multiply each midpoint by its frequency, sum all, divide by total frequency.
Assumed Mean Method (Shortcut)
x̄ = a + Σ(fᵢdᵢ) / Σfᵢ    where dᵢ = xᵢ − a
a = assumed mean (usually the middle class midpoint). Reduces arithmetic by working with smaller numbers.
Step Deviation Method
x̄ = a + (Σfᵢuᵢ / Σfᵢ) × h    where uᵢ = dᵢ / h
h = class width. Further simplifies when class width is uniform.

📐 Median & Mode of Grouped Data

Median: Median = l + [(n/2 − cf) / f] × h

Where: l = lower class boundary of median class, n = total frequency, cf = cumulative frequency before median class, f = frequency of median class, h = class width.

Median class: The class whose cumulative frequency first exceeds n/2.

Mode: Mode = l + [(f₁ − f₀) / (2f₁ − f₀ − f₂)] × h

Where: l = lower boundary of modal class, f₁ = frequency of modal class, f₀ = frequency before modal class, f₂ = frequency after modal class, h = class width.

Modal class: The class with the highest frequency.

Find the mode. Modal class 20–30. f₁=12, f₀=8, f₂=7, l=20, h=10.

5

1

2

⚠️ Watch Out!

  • MIDPOINTThe class midpoint xᵢ = (lower boundary + upper boundary) / 2. Use this, not either boundary alone.
  • cf ORDERCumulative frequency is running total — add frequencies one class at a time from the top down.
  • MEDIAN CLASSThe median class is where cumulative frequency first reaches or exceeds n/2, not exactly equals it.
  • EMPIRICALMode = 3 Median − 2 Mean is an empirical approximation, not an exact formula. Only use for estimation.
