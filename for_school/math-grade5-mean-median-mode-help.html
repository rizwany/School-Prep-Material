Mean, Median & Mode

Grade 5 Math  ·  Averages and data analysis

📊 Four Measures of Data

Mean (Average)

Add all values, then divide by the count. Most commonly used average.

Median (Middle)

Sort the values. The middle value is the median. If two middle values, take their mean.

Mode (Most Common)

The value that appears most often. A data set can have 0, 1, or more modes.

Range (Spread)

Largest value minus smallest value. Shows how spread out the data is.

📐 Formulas

Mean
Mean = (sum of all values) ÷ (number of values)
Median — odd count
Sort values. Median = middle value
Median — even count
Sort values. Median = (left-middle + right-middle) ÷ 2
Range
Range = largest value − smallest value

⚠️ Watch Out!

  • SORT FIRST Always sort the data before finding the median. Forgetting this is the most common mistake.
  • EVEN COUNT With an even number of values, the median is the mean of the two middle numbers — not one of them.
  • NO MODE If all values appear equally often, there is no mode. Don't invent one!
  • COUNT VALUES For the mean, count how many values there are carefully — it's easy to miscount.
