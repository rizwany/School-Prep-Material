Basic Measurement

In this guide, you'll learn to compare objects by length, weight, and how much they hold, using words like longer, shorter, heavier, lighter, and more.

⏱ ~5 min read 📝 Jump to Worksheet → اردو گائیڈ

Section 1

What is Measurement?

Measurement means finding out how big, heavy, or full something is. In Grade 1, we compare two or three things and use describing words to say which is bigger, heavier, or holds more.

We compare length (how long/tall), weight (how heavy), and capacity (how much something holds).

Always line up the start points before comparing length. Use your hands or a balance scale to compare weight.

Section 2

How to Compare

  1. 1
    Length: Line both objects up at the same starting point. The one that sticks out further is longer.
  2. 2
    Weight: Hold one object in each hand, OR use a balance scale. The side that goes down is heavier.
  3. 3
    Capacity: Pour each container into the same sized cup and count how many cups it fills. More cups = holds more.
  4. 4
    Non-standard units: Use paper clips, hand spans, or footsteps to measure. Count how many fit — more units = longer.
  5. 5
    Ordering 3 objects: Compare the shortest pair first to find the shortest, then find the longest.

Section 3

Worked Examples

Example 1 — Compare length
A ribbon is 8 paper clips long. A rope is 5 paper clips long. Which is longer?
Ribbon: 8 clips. Rope: 5 clips.
8 > 5, so the ribbon is longer.
✅ Answer: The ribbon is longer
Example 2 — Compare weight
A feather and a stone are placed on a balance. The stone side goes down. Which is heavier?
The side that goes down is heavier.
The stone side went down → stone is heavier.
✅ Answer: The stone is heavier
Example 3 — Compare capacity
A mug fills 3 cups of water. A bottle fills 7 cups. Which holds more?
Mug: 3 cups. Bottle: 7 cups.
7 > 3, so the bottle holds more.
✅ Answer: The bottle holds more
Example 4 — Order 3 objects
Order from shortest to longest: ruler (10 clips), pencil (7 clips), eraser (3 clips).
3 clips < 7 clips < 10 clips
eraser < pencil < ruler
✅ Answer: Eraser, pencil, ruler
Example 5 — Word problem
Mia's arm span is 4 hand spans wide. Ali's arm span is 5 hand spans wide. Whose arm span is wider?
Mia: 4 spans. Ali: 5 spans.
5 > 4, so Ali's arm span is wider.
✅ Answer: Ali's arm span is wider

Section 4

Watch Out! ⚠️

Common Mistakes to Avoid

  • ❌ Not lining up objects at the same start: Comparing length when one object starts further forward
    ✅ Always start both objects from the same line. Use a flat edge or wall to align them.
  • ❌ Confusing taller with heavier: Thinking the taller bottle holds more
    ✅ A short wide container can hold MORE than a tall thin one. Always test capacity directly, don't judge by height alone.
  • ❌ Using different sized non-standard units: Measuring with big paper clips then small ones
    ✅ Always use the same sized units for the whole measurement. Switching unit size gives a wrong count.
