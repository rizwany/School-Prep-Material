Statistics & Frequency Tables

Grade 6 Math  ·  Organising and interpreting data

📊 Key Concepts

Frequency

How many times a value appears in a data set. Counted using tally marks.

Frequency Table

A table showing each value and how often it occurs. Always check totals.

Tally Marks

4 vertical lines + 1 diagonal cross = 5. Groups of 5 make counting easy.

Relative Frequency

Frequency ÷ Total. Shows each value as a fraction or percentage of the whole.

Tally Mark Groups
| = 1    || = 2    ||| = 3    |||| = 4    ꟷ|||| = 5
Key Formulas
Total frequency = sum of all frequencies
Relative frequency = frequency ÷ total
Mode = value with the highest frequency

⚠️ Watch Out!

  • CHECK TOTALS Always verify that all frequencies add up to the total. If they don't, you've made an error.
  • TALLY GROUPS Count tally marks in groups of 5. Remember the 5th mark is a diagonal line through the 4 upright ones.
  • MODE ≠ MEAN The mode is the most frequent value, not the average. A data set can have more than one mode.
  • RELATIVE FREQ Relative frequency is always between 0 and 1 (or 0% and 100%). If you get something outside this range, recheck your calculation.
