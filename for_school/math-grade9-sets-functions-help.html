Sets & Functions

Grade 9 Math  ·  Set notation, operations & function basics

📦 What is a Set?

A set is a well-defined collection of distinct objects, called elements or members. Sets are written with curly braces: A = {1, 2, 3, 4}.

n(A) denotes the number of elements in set A (its cardinality).

Special sets: ∅ or {} = empty set (no elements). U = universal set (all elements under consideration).

Notation: 3 ∈ A means "3 is an element of A". 5 ∉ A means "5 is NOT in A".

∩ Set Operations

A ∪ B
Union
All elements in A OR B (or both)
A ∩ B
Intersection
Elements in BOTH A AND B
A′ or Aᶜ
Complement
Elements in U but NOT in A
A − B
Difference
Elements in A but NOT in B

Key formula (inclusion-exclusion): n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

◯ Venn Diagram

→ Functions

A function is a rule that assigns exactly one output (y-value) to each input (x-value). Written as f: A → B or f(x) = ….

Domain
The set of all valid input values (x-values). Every element must be mapped.
Codomain
The set of all possible output values declared for the function.
Range
The set of ACTUAL output values produced by the function (subset of codomain).
Vertical Line Test
A graph represents a function if every vertical line crosses the graph at most once.

✏️ Worked Examples

1

A = {1,2,3,4,5}, B = {3,4,5,6,7}. Find A ∪ B and A ∩ B.

1

A ∪ B = all elements in either set: {1, 2, 3, 4, 5, 6, 7}.

2

A ∩ B = elements in both: {3, 4, 5}.

A ∪ B = {1,2,3,4,5,6,7}  |  A ∩ B = {3,4,5}
2

In a class of 40: 25 like Maths (M), 20 like Science (S), 10 like both. How many like neither?

1

n(M ∪ S) = n(M) + n(S) − n(M ∩ S) = 25 + 20 − 10 = 35.

2

Neither = Total − n(M ∪ S) = 40 − 35 = 5.

5 students like neither
3

U = {1,2,3,4,5,6,7,8}, A = {2,4,6,8}. Find A′.

1

A′ = all elements in U that are NOT in A.

2

A′ = {1, 3, 5, 7}.

A′ = {1, 3, 5, 7}
4

f(x) = 3x − 1. Find f(4) and f(−2).

1

f(4) = 3(4) − 1 = 12 − 1 = 11.

2

f(−2) = 3(−2) − 1 = −6 − 1 = −7.

f(4) = 11, f(−2) = −7
5

Is the mapping {(1,2), (2,4), (3,2), (4,6)} a function?

1

Check: each input (1, 2, 3, 4) maps to exactly one output.

2

1→2, 2→4, 3→2, 4→6. No input has two different outputs (3 and 1 both mapping to 2 is fine — outputs can repeat).

Yes, this is a function

⚠️ Watch Out!

  • DISTINCTSets contain no duplicates: {1, 2, 2, 3} is the same as {1, 2, 3}. List each element once only.
  • FUNCTIONA relation is NOT a function if any input maps to MORE THAN ONE output. The input 2 → 3 and 2 → 5 fails.
  • RANGE ≠ CODOMAINThe range is the set of values actually produced, which may be a proper subset of the codomain.
  • ∅ ≠ {∅}The empty set ∅ has 0 elements. The set {∅} has 1 element (the empty set itself).
