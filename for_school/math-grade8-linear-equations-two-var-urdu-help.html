دو متغیر خطی مساواتیں

جماعت ۸ | Linear Equations in Two Variables

حصہ ۱: بیک وقت مساواتیں

دو مساواتیں جن میں دو متغیر ہوں — ایک ہی وقت میں دونوں مساواتیں پوری کرنے والی x اور y قیمتیں تلاش کرنا۔

ax + by = c اور dx + ey = f

حصہ ۲: تبدیلی طریقہ (Substitution)

مثال: x + y = 5 اور x − y = 1

۱
پہلی مساوات سے: x = 5 − y
۲
دوسری میں رکھیں: (5−y) − y = 1 → 5−2y = 1 → y = 2
۳
x = 5−2 = 3 → جواب: x=3، y=2

حصہ ۳: خاتمہ طریقہ (Elimination)

مثال: 2x + y = 7 اور x + y = 4

۱
پہلی منہا دوسری: (2x−x) + (y−y) = 7−4 → x = 3
۲
x=3 رکھیں: 3+y=4 → y=1
۳
جواب: x=3، y=1

حصہ ۴: یاد رکھیں
