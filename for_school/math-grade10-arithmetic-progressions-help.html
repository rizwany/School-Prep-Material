Arithmetic Progressions

Grade 10 Math  ·  nth term, sums & AP properties

🔢 What is an AP?

An Arithmetic Progression (AP) is a sequence where each term after the first is obtained by adding a fixed number called the common difference (d).

General form: a, a+d, a+2d, a+3d, … where a is the first term and d can be positive, negative, or zero.

📐 Key Formulas

nth Term
aₙ = a + (n − 1)d
a = first term, d = common difference, n = term number
Sum (n terms)
Sₙ = n/2 × [2a + (n − 1)d]
Use when a and d are known
Sum (first & last)
Sₙ = n/2 × (a + l)
l = last term. Use when first and last terms are known
Common Difference
d = aₙ − aₙ₋₁
Subtract any term from the next term

✏️ Worked Examples

1

Find the 15th term of the AP: 4, 7, 10, 13, …

1

a = 4, d = 7 − 4 = 3, n = 15.

2

a₁₅ = 4 + (15−1)×3 = 4 + 42 = 46.

15th term = 46
2

Find the sum of the first 20 terms of 2, 5, 8, 11, …

1

a = 2, d = 3, n = 20.

2

S₂₀ = 20/2 × [2(2) + 19×3] = 10 × [4 + 57] = 10 × 61 = 610.

S₂₀ = 610
3

The 3rd term of an AP is 12 and the 7th term is 24. Find a and d.

1

a₃ = a + 2d = 12  and  a₇ = a + 6d = 24.

2

Subtract: 4d = 12 → d = 3.

3

a = 12 − 2(3) = 6.

a = 6, d = 3
4

Which term of the AP 5, 11, 17, … is 95?

1

a = 5, d = 6. Set aₙ = 95: 5 + (n−1)×6 = 95.

2

(n−1)×6 = 90 → n−1 = 15 → n = 16.

95 is the 16th term
5

Sum of first n terms of an AP is 3n² + 5n. Find the 8th term.

1

aₙ = Sₙ − Sₙ₋₁ for n ≥ 2.

2

S₈ = 3(64)+5(8) = 192+40 = 232. S₇ = 3(49)+5(7) = 147+35 = 182.

3

a₈ = 232 − 182 = 50.

8th term = 50

⚠️ Watch Out!

  • n STARTS AT 1In aₙ = a + (n−1)d, n is the position (1st term, 2nd term…), not an index starting at 0.
  • NEGATIVE dAn AP can decrease: 20, 15, 10, … has d = −5. Always subtract consecutive terms to find d.
  • SUM vs TERMSₙ is the sum of the first n terms. aₙ is the nth term. Do not confuse them.
  • a₁ vs aThe first term is written as a or a₁. Both mean the same thing.
