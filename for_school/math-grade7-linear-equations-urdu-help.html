خطی مساوات

خطی مساوات وہ مساوات ہے جس میں متغیر (x) کی قوت ۱ ہو۔ ax + b = c کی شکل میں x کو تنہا کرنا سیکھیں۔

⏱ ~۵ منٹ مطالعہ 📝 مشق پر جائیں ←

حصہ ۱

خطی مساوات کیا ہے؟

مساوات ایک ریاضی جملہ ہے جس میں دونوں طرف کی قدریں برابر ہوتی ہیں۔ خطی مساوات کی عام شکل ہے:

ax + b = c

یہاں x وہ متغیر ہے جسے ہم تلاش کرنا چاہتے ہیں۔ a، b اور c معلوم اعداد ہیں۔

مثلاً: 2x + 3 = 11 — یہاں a=2، b=3، c=11 ہے اور x کی قدر تلاش کرنی ہے۔

حصہ ۲

مساوات حل کرنے کا طریقہ

  1. ۱
    مساوات پڑھیں۔ ax + b = c کی شکل پہچانیں۔
  2. ۲
    b کو دونوں طرف سے منہا کریں۔ ax + b − b = c − b → ax = c − b
  3. ۳
    a سے تقسیم کریں۔ ax ÷ a = (c−b) ÷ a → x = (c−b)/a
  4. ۴
    جواب چیک کریں۔ x کی قدر اصل مساوات میں رکھ کر تصدیق کریں۔

حصہ ۳

حل شدہ مثالیں

مثال ۱ — x + 5 = 12
x + 5 = 12 میں x کتنا ہے؟
x + 5 = 12
x = 12 − 5 (دونوں طرف سے 5 گھٹائیں)
x = 7
✅ x = 7
مثال ۲ — 3x = 18
3x = 18 میں x کتنا ہے؟
3x = 18
x = 18 ÷ 3 (دونوں طرف کو 3 سے تقسیم کریں)
x = 6
✅ x = 6
مثال ۳ — 2x + 3 = 11
2x + 3 = 11 میں x کتنا ہے؟
2x + 3 = 11
2x = 11 − 3 = 8 (3 منہا کریں)
x = 8 ÷ 2 = 4 (2 سے تقسیم)
✅ x = 4
مثال ۴ — 5x − 7 = 13
5x − 7 = 13 میں x کتنا ہے؟
5x − 7 = 13
5x = 13 + 7 = 20 (7 جمع کریں)
x = 20 ÷ 5 = 4
✅ x = 4
مثال ۵ — لفظی مسئلہ
ایک عدد کو تین گنا کر کے ۵ جمع کریں تو ۲۰ ملتا ہے۔ وہ عدد کیا ہے؟
3x + 5 = 20
3x = 15 → x = 5
✅ وہ عدد 5 ہے

حصہ ۴

احتیاط کریں! ⚠️

عام غلطیاں

  • ❌ علامت بھول جانا: 2x − 3 = 7 میں 2x = 7 − 3 لکھنا
    ✅ 2x = 7 + 3 = 10 (منفی ہو تو جمع ہوگا)
  • ❌ صرف ایک طرف عمل: 2x + 4 = 10 میں صرف بائیں سے 4 گھٹانا
    ✅ دونوں طرف سے یکساں عمل: 2x + 4 − 4 = 10 − 4
  • ❌ چیک نہ کرنا: جواب نکال کر مساوات میں نہ لگانا
    ✅ ہمیشہ x کی قدر اصل مساوات میں رکھ کر تصدیق کریں

حصہ ۵

فوری حوالہ

مساوات

متغیر (x)
وہ نامعلوم عدد جسے تلاش کرنا ہے۔
تنہا کرنا

جمع ↔ منہا، ضرب ↔ تقسیم۔
توازن
ایک طرف جو عمل ہو دوسری طرف بھی وہی۔
